# 自定义状态栏图标

# 版本要求

0.8.9 版本后，支持自定义状态栏图标

# 配置

  1. 进入 Settings 界面
  2. 滚动至 Appearance 栏
  3. 在 Default Icon Path 及 System Proxy On Icon Path 右边输入栏中输入图标路径

# ICO 文件大小

建议 64x64，如果不清晰可以尝试 32x32
